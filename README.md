# Data Analyst Portfolio

A modern, responsive portfolio website designed for data analysts.

## Features
- **Dark Mode Design**: Premium aesthetic with glassmorphism and animated backgrounds.
- **Responsive**: Works on desktop, tablet, and mobile.
- **Sections**:
  - **Hero**: Introduction and value proposition.
  - **About**: Bio and highlighted skills.
  - **Skills**: Technical stack (Python, SQL, Tableau, etc.).
  - **Projects**: Case study cards with hover effects.
  - **Contact**: Simple contact form layout.

## How to Use
1. **Open**: Double-click `index.html` to view the website in your browser.
2. **Customize**:
   - Edit `index.html` to update your name, bio, and project details.
   - Replace the FontAwesome icons in `<i>` tags with your preferred icons.
   - Update `style.css` variables (`--accent-primary`, `--accent-secondary`) to change the color scheme.
3. **Images**:
   - Currently, the site uses CSS gradients and placeholders. To add real images:
     - Save your images in an `images/` folder.
     - Update the `src` attributes in `index.html` or background images in `style.css`.

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- JavaScript (Vanilla)
